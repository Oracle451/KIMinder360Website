KIMinder360 Mobile App

The KIMinder360 App trains users in crucial Situational Awareness skills through the sending out of mindfullness directives during action windows throughout the day. It allows for customization to match personal preferences and provides an end-of-day evaluation and tracking of progress. Our goal is to make Situational Awareness training accessible, fun, and effective for everyone.

Features & Functionality

The KIMinder360 App allows users to select “Action Windows” for receiving notifications and to choose from multiple categories of directives, including personal safety, tactical readiness, fitness, social interactions, and more. Users can track their performance via the app dashboard.

Key Features:

  • Randomized directives for situational awareness and self-improvement.
  • Customizable action windows and category selections.
  • End-of-day progress evaluation and feedback.
  • Dashboard for tracking performance and statistics.
  • Expandable database of exercises with future user contributions.

Technologies & Tools

We are using Android Studio for mobile development, cloud databases for user data storage, notification services for directive delivery, and UI/UX best practices to ensure usability. Future expansions may include cross-platform frameworks for iOS deployment.
